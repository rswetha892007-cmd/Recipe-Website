🍴 Tasty Recipes

A modern, responsive recipe website developed using HTML5, CSS3, and JavaScript. The project provides a clean, user-friendly interface for exploring delicious recipes with an attractive design and responsive layout.

---

📖 Project Overview

Tasty Recipes is a front-end web development project that showcases a collection of popular recipes, including Pizza, Biryani, and Cake. Each recipe has its own dedicated page featuring ingredients, preparation steps, and an elegant user interface designed to provide a pleasant browsing experience.

---

✨ Features

- Responsive and modern user interface
- Beautiful hero section with background image
- Professional navigation bar
- Recipe cards with hover effects
- Dedicated pages for each recipe
- Well-structured Ingredients and Steps sections
- Attractive buttons and animations
- Responsive layout for different screen sizes
- Professional footer
- Clean and organized design

---

🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
